## Installation

- Navigate to Project Directoy

```bash
cd path/super-app-spin
```

- Install dependencies

```bash
yarn install
```

## Usage

- Clone this repo.
- Run `nvm use` to use the node version specified in the `.nvmrc` file.

```bash
nvm use
```

- Install the dependencies.

```bash
yarn
```

- Start the project.

```bash
yarn run ios
yarn run android
```

- Start db.json

```bash
yarn run json-server
```
